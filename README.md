## 🎯 Зачем этот проект?

**CRYPTO-GONOX** — это учебная и развлекательная платформа, где каждый может:

* ✅ Попробовать себя в трейдинге **без риска потери реальных денег**.
* 🧠 Понять механику рынка и протестировать свои инвестиционные решения.
* 📊 Увидеть наглядный результат своих действий в цифрах и графиках.
* 🏆 Сравнить свой успех с достижениями других участников в таблице лидеров.

---

## ⚙️ Как это работает

1. 👋 **Старт:** Новый пользователь регистрируется и получает стартовый виртуальный баланс.
2. 📈 **Рынок:** Цены криптовалют подтягиваются в реальном времени.
3. 🤝 **Трейдинг:** Пользователь покупает и продает активы, формируя свой портфель.
4. 📝 **Аналитика:** Все сделки сохраняются в истории для детального анализа.
5. 🥇 **Соревнование:** Итоговый капитал определяет место пользователя в глобальном лидерборде.

---

## ✅ Реализованные функции

* 🔐 **Авторизация:** Безопасный вход через Firebase Auth.
* 💰 **Виртуальный кошелек:** Учет баланса и стоимости портфеля.
* 🔄 **Транзакции:** Симуляция покупки/продажи по актуальному курсу.
* 📜 **История:** Подробный лог всех торговых операций.
* 👤 **Социальность:** Просмотр профилей и портфелей других трейдеров.
* 📊 **Рейтинг:** Глобальная таблица лидеров.
* 📈 **Графики:** Визуализация движения цен на активы.
* 📱 **Адаптивность:** Полностью адаптивный интерфейс для ПК и смартфонов.

## 🚀 Запуск проекта локально

Инструкция для развертывания проекта на вашем компьютере:

```bash
# 1. Клонируйте репозиторий
git clone [https://github.com/DEZERIandroid/CRYPTO-GONOX.git](https://github.com/DEZERIandroid/CRYPTO-GONOX.git)

# 2. Перейдите в директорию проекта
cd CRYPTO-GONOX

# 3. Установите зависимости
npm install

# 4. Запустите dev-сервер
npm run dev

```

## 🔥 Настройка Firebase

Для работы облачных функций, аутентификации и базы данных необходимо привязать проект к вашему инстансу Firebase.

1. Создайте файл `.env` в корневой директории проекта.
2. Добавьте в него следующие переменные (замените значения на свои из консоли Firebase):

```env
  VITE_FIREBASE_API_KEY=your_api_key
  VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
  VITE_FIREBASE_PROJECT_ID=your_project_id
  VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
  VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
  VITE_FIREBASE_APP_ID=your_app_id

[!TIP]
Все эти ключи можно найти в вашей консоли Firebase: Project Settings (иконка шестеренки) ➡️ General ➡️ раздел Your apps.


```
